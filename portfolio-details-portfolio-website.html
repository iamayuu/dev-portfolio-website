Project Overview

This personal portfolio website is a lightweight, responsive front end that showcases projects, carrer timeline, and contact options. The site is built with semantic HTML, custom CSS that follows a clean aesthetic, and small vanilla JavaScript modules for interactive elements such as counters, the skill chips, and the contact form.

The goal was to keep the site fast, accessible, and easy to maintain while providing a professional presentation for recruiters and clients.

Key Features

  • Responsive grid layout and accessible semantics for faster scanning.
  • Contact form integrated with Formspree for static hosting email delivery.
  • Animated skill chips with crisp hover interaction and clipboard copy for contact details.
  • Portfolio grid with filter support and lightbox previews for project screenshots.
  • Performance focused: minimal dependencies, optimized assets, and responsive images.
